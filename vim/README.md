# vimrc

## Keybinds

- gd Go definition
- Ctrl + t Open in new tab NERDTree
- , Is Leader key
- , For emmet completion
- Shift + c + Space to comment line
- :ser <search> To search project wide
- F8 Previous tab
- F9 Next tab
- :tabe <filename> Open file in new tab
- :tab ter Open terminal in new tab
